Uniswap provides a family of open-source protocols that power decentralized trading, liquidity provisioning, and onchain order execution.

| Protocol | Purpose |
|---|---|
| [Uniswap v4](/docs/protocols/v4/overview) | Concentrated liquidity AMM with hooks for custom pool logic |
| [Uniswap v3](/docs/protocols/v3/overview) | Concentrated liquidity AMM |
| [Uniswap v2](/docs/protocols/v2/overview) | Constant-product AMM |
| [UniswapX](/docs/protocols/uniswapx/overview) | Dutch-auction order protocol for optimal execution |
| [Universal Router](/docs/protocols/universal-router/overview) | Flexible calldata router for multi-protocol swaps |
| [Permit2](/docs/protocols/permit2/overview) | Shared token-approval and transfer system |
