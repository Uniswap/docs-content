# Trading API

The Uniswap Trading API provides a simple interface for executing swaps and managing liquidity positions.

## Endpoints

- **[Swapping](/docs/api/endpoints/swapping)** - Quote and execute token swaps
- **[Liquidity Provisioning](/docs/api/endpoints/liquidity-provisioning)** - Create and manage LP positions
- **[Reference Data](/docs/api/endpoints/reference-data)** - Token and pool information
- **[Utility](/docs/api/endpoints/utility)** - Helper endpoints for calldata and approvals

## OpenAPI Specification

The full OpenAPI specification is available at [/docs/api/openapi.json](/docs/api/openapi.json).
