## Chainlink

Chainlink [Data Feeds](https://data.chain.link/) are the market-leading provider of secure, reliable data to smart contracts across multiple blockchains. Independent data providers contribute raw pricing and market data, which is aggregated by a decentralized set of Chainlink nodes into tamper-resistant oracle reports. These feeds power a broad range of financial protocols that depend on accurate market data to execute correctly and safely.

- Tamper-resistant, Sybil-resistant data aggregation

- Transparent node participation and feed monitoring
- Standardized interfaces for easy, cross-chain integration
