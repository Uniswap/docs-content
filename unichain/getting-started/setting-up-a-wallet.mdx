## Access Unichain with Uniswap Wallet
Use the Uniswap Wallet to interact with Unichain and Unichain Sepolia seamlessly. The Uniswap Wallet is a self-custodial crypto wallet built for swapping. It supports 12+ chains, including testnets, and lets users swap tokens on Uniswap, browse NFT collections, and transact on dApps while maintaining full control of their assets. It is available on Chrome, iOS, and Android. [Learn more](https://wallet.uniswap.org/).

To use the Uniswap Wallet on Unichain Sepolia:

1. Download the wallet
    - [Download on Chrome](https://chromewebstore.google.com/detail/uniswap-extension/nnpmfplkfogfpmcngplhnbdnnilmcdcg)
    - [Download on iOS](https://uniswapwallet.onelink.me/HcBh/27c373zb)
    - [Download on Android](https://uniswapwallet.onelink.me/vhxJ/tj22hom3)
2. Press the settings icon on the top right of your screen.
3. Then, turn the `Testnet mode` toggle on. Now you're automatically connected to Unichain Sepolia Testnet and can view any balances, sign testnet transactions, and swap testnet assets.

## Add to an Existing Wallet

You can add Unichain support to your existing Ethereum wallet by using the "Add Network" flow and entering the following network information:

### Unichain Mainnet
| **Chain Name** | Unichain Mainnet |
| --- | --- |
| **Chain ID** | 130 |
| **RPC URL** | https://mainnet.unichain.org (rate limited, not for production) |
| **Currency Symbol** | ETH |
| **Block Explorer** | https://uniscan.xyz |

### Unichain Sepolia Testnet
| **Chain Name** | Unichain Sepolia |
| --- | --- |
| **Chain ID** | 1301 |
| **RPC URL** | https://sepolia.unichain.org (rate limited, not for production)|
| **Currency Symbol** | ETH |
| **Block Explorer** | https://sepolia.uniscan.xyz/ |
