# Uniswap Documentation

Welcome to the Uniswap documentation. Here you'll find guides and references for building on Uniswap.

## Getting Started

New to Uniswap? Start with our [overview](/docs/get-started) to understand the protocol fundamentals.

## Explore

- **[Get Started](/docs/get-started)** - Learn the basics of AMMs, liquidity, and trading. Covers core concepts like swaps, pools, and protocol architecture.
- **[Trading](/docs/trading)** - Guides for executing swaps. Includes quickstarts for v2, v3, and v4 as well as advanced patterns like flash swaps and multi-hop routing.
- **[Liquidity](/docs/liquidity)** - Provide liquidity and earn fees. Covers pool creation, position management, and UniswapX filling.
- **[Protocols](/docs/protocols)** - Deep dives into v4, v3, v2, and more. Technical references for smart contracts, hooks, and protocol internals.
- **[SDKs](/docs/sdks)** - JavaScript/TypeScript libraries for integration. Build frontends and bots that interact with Uniswap pools.
- **[API](/docs/api)** - Trading API reference and guides. REST endpoints for quoting, swapping, and managing liquidity positions.
